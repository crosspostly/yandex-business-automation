# Task Schedule: Agent Operation Blueprint 2026

This document defines when and why each specialized agent persona is triggered.

## 🏛 Tier 1: Foundation (Setup Only - Run Once)
*Trigger: Initial onboarding or rebranding.*

### 1. SEO Specialist (Architect Mode)
- **Tasks**: Keyword research, Primary/Secondary Rubric selection, Short Name optimization.
- **Why**: These are the "anchors" of your entity. Changing them too often can trigger temporary ranking drops.

### 2. Media Designer (Gallery Mode)
- **Tasks**: Uploading "Entrance", "Interior", and "Exterior" photos with GPS geo-tags.
- **Why**: Yandex needs physical proof of location. Once verified, these rarely change unless the office moves.

### 3. Technical Director
- **Tasks**: Map Widget extraction, Site generation trigger, Initial Delivery settings.
- **Why**: Integration steps for external websites.

---

## 🔄 Tier 2: Maintenance (Recurring - Scheduled)
*Trigger: Cron jobs or periodic manual triggers.*

### 1. Content Manager (Daily/Weekly)
- **Daily**: **Stories** (Maintenance of the +15% ranking boost).
- **Weekly**: **Publications** (SEO articles/posts with keywords).
- **Monthly**: **Events & Promotions** (Conversion optimization).

### 2. Reputation Manager (Daily/As Needed)
- **Tasks**: Responding to reviews, Monitoring rating trends.
- **Why**: Faster response time = Higher trust score.

### 3. SEO Specialist (Updater Mode - Monthly)
- **Tasks**: Updating Price List (YML) with new LSI keywords based on search trends.
- **Why**: To keep the 'Goods' search relevant to current user queries.

### 4. Media Designer (Dynamic Mode - Bi-Weekly)
- **Tasks**: Fresh photos of "Equipment" or "Products" to show active business state.
